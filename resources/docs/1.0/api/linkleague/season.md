## ink League Season Standing

### Endpoint

|Method|URI|Headers|
|:-|:-|:-|
|POST|`/api/leagues/{division}/info/league_standings/filter`|`Bearer Token`|

### URL Params

|Param|Type|Values|Example
|:-|:-|:-|:-
|`division`|`integer`|`valid numeric division id`|Ex: `1`
|`filter`|`string`|`fixed string`|Ex: `season`
|`linkedLeague`|`integer`|`valid numeric parent linked league id`|Ex: `1`

> {success} Success Response

Code `200`
