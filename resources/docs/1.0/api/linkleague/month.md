## Link League Month Standing

### Endpoint

|Method|URI|Headers|
|:-|:-|:-|
|POST|`/api/leagues/{division}/info/league_standings/filter`|`Bearer Token`|

### URL Params

|Param|Type|Values|Example
|:-|:-|:-|:-
|`division`|`integer`|`required`|Ex: `1`
|`startDate`|`Date`|`required`|Ex: `2019-05-01 (Y-m-d)`
|`endDate`|`Date`|`required`|Ex: `2019-05-30 (Y-m-d)`
|`linkedLeague`|`integer`|`required`|Ex: `1`

> {success} Success Response

Code `200`
